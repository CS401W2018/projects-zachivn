Internet

The Internet is a worldwide network of computers and devices that use standard communication protocols like TCP/IP to talk to each other.

Web

The Web is a system of information accessed through the Internet.

How are the Internet and Web different?

The Internet is the global infrastructure of computers and devices, while the Web is a service on the Internet that users access and interact with web pages.

ASCII and Unicode

ASCII is a character encoding standard that uses numbers from 0 to 127 to represent text in computers, limited to 128 characters.

What is UTF-8

UTF-8 is a variable-length character encoding system for Unicode, using 1 to 4 bytes to represent each character.

URL

A URL is the address used to access a resource on the Web.

Domain Name

A domain name is the human-readable address of a website.

HTTP

HTTP is a protocol for transferring web pages on the Web.

HTTPS

HTTPS is the secure version of HTTP, using encryption to protect data.

IP Address

An IP address is a unique label for devices on a network, allowing them to communicate.

DNS

DNS translates domain names into IP addresses.

W3C Recommendations

W3C Recommendations are standards for web technologies.

XML, XHTML, HTML, HTML5

- XML: A flexible language for storing and transporting data. - XHTML: A stricter version of HTML that follows XML rules. - HTML: The standard language for creating web pages. - HTML5: The latest version of HTML with new features.
